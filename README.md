# 💅 CG Beauty Studio - Website Generation Project

## 📖 Sobre o Projeto
Este repositório documenta o processo de ideação, engenharia de prompts e design conceptual para a criação de uma *Landing Page* para o **Cátia Gomes Beauty Studio** (um estúdio focado em estética de unhas e nail art). 

O objetivo é estruturar as informações recolhidas das redes sociais da marca e fornecer os recursos necessários para implementar o website de forma rápida e gratuita, utilizando geradores de Inteligência Artificial ou plataformas *No-Code*.

## 🎨 Identidade Visual e Branding
* **Design:** Minimalista, elegante, focado em imagens de alta qualidade.
* **Paleta de Cores:** Verde menta suave, off-white, creme/nudes e detalhes em dourado.
* **Tom de Voz:** Profissional, confiável e focado no bem-estar, segurança e higiene.

## 🛠️ Recursos Desenvolvidos
Este projeto contém os elementos base para a construção do site:

1.  **Engenharia de Prompt (AI Builders):** Prompts detalhados e otimizados (versão completa e versão reduzida de ~500 caracteres) prontos a ser introduzidos em plataformas de geração de sites (ex: Framer AI, Wix Studio).
2.  **Copywriting:** Textos persuasivos para as secções principais (Sobre, Serviços, Testemunhos, Segurança).
3.  **Wireframe Estrutural:** Esboço do layout da página única (*Single-Page*), desenhado para guiar a experiência da utilizadora até ao agendamento.
4.  **SEO (Meta Tags):** *Meta descriptions* otimizadas para os resultados do Google e partilha em redes sociais/WhatsApp (focadas em conversão local na Guarda, Portugal).

## 🚀 Ferramentas de Implementação Recomendadas
Para colocar este projeto online de forma 100% gratuita, recomenda-se a utilização das seguintes plataformas:
* **Framer (AI):** Para um design altamente moderno, com geração automática através do prompt.
* **Wix:** Para opções focadas em negócios locais, com integração fácil de sistemas de agendamento online.
* **Canva Websites:** Para uma implementação manual rápida, intuitiva e totalmente gratuita usando moldes visuais.
